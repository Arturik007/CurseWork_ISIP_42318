Как скачать, установить и запустить сайт «Склад WMS» на Windows 10

Что у вас есть:

* Проект в папке: c:\Users\Путь к папке с проектом 
* Приложение в подпапке: warehouse
* Главный файл запуска: warehouse\app.py
* Сайт открывается в браузере по адресу: http://127.0.0.1:5000

Что нужно установить (делается один раз):

1. Python
* Скачайте Python версии 3.10 или выше (рекомендуется 3.11 или 3.12).
* В установщике поставьте галочку: "Add Python to PATH".
* Проверьте установку:

  python --version
  pip --version

  Если команды не найдены, значит Python не добавлен в PATH.
2. (Опционально) Git
* Git нужен, если вы будете скачивать проект из интернета через git clone.
* Если проект уже на компьютере, то Git не нужен.

Как скачать проект (если его нет на компьютере):

1. Вариант A — ZIP
* Скачайте архив ZIP с GitHub или другого места.
* Распакуйте архив в папку, например: c:\Users\ROMAN\Desktop\курсовая.
2. Вариант B — Git
* Откройте PowerShell.
* Введите команды:

  git clone <адрес_репозитория>
  cd <папка_проекта>

Установка зависимостей (пакетов) — «скачать то, что нужно сайту»:

* Зависимости описаны в файле warehouse\requirements.txt.
* Для установки рекомендуется использовать виртуальное окружение (venv):
  * Откройте PowerShell в корне проекта: c:\Users\ROMAN\Desktop\курсовая.
  * Введите команды:

    python -m venv .venv
    python -m pip install --upgrade pip
    pip install -r requirements.txt

Запуск сайта:

* Откройте PowerShell в корне проекта.
* Введите команду:

  python app.py
* В консоли появится сообщение, что сервер запущен.
* Откройте в браузере: http://127.0.0.1:5000.
* По умолчанию логин и пароль: admin.
